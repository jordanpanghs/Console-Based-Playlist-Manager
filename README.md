# Song Playlist Manager in C++

### This is my first ever coding project! 

This project was a group assignment for our first coding class of our Software Engineering degree.

Options Include:
- Adding a Song
- Removing a Song
- Edit a Song
- List Songs
- Search Song
- Shuffling Playlist

Playlist will save first before exiting.
